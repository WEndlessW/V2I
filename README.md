## Pages

Tato stánka má 3 HTML soubory:

| Soubor | Popis |
| --- | --- |
| `index.html` | Domovská stánka s uvítáním |
| `characters.html` | Stánka s tabulkou nejslavnějších anime charakterů |
| `contact.html` | Stánka s různými typy inputů |

Všechny stránky používají ten stejný stylesheet: `styles.css`.

## Funkce

- Hlavička a patička na každé stránce
- Navigační menu které linkuje všechny tři stránky a poté také na externí stránku (Crunchyroll)
- Nadpisy: `h1`, `h2`, and `h3`
- Obrázek
- Tabulku
- Formulář s různými typy inputů

## Struktura souborů

```
├── characters.html
├── contact.html
├── styles.css

```
